# Быстрый старт - My POS System

## Что уже готово (MVP)

✅ **Backend API** - FastAPI + SQLite
✅ **Интерфейс кассы** - для планшета (сенсорный)
✅ **Админка** - управление товарами
✅ **Дашборд** - статистика в реальном времени

---

## Запуск проекта

### 1. Backend (API сервер)

```bash
cd /Users/Dom/MyPosSystem/backend

# Активировать виртуальное окружение
source venv/bin/activate

# Запустить сервер
./start.sh

# Или вручную:
# uvicorn main:app --reload
```

Backend будет доступен на:
- API: http://localhost:8000
- Документация: http://localhost:8000/docs

### 2. Frontend (Веб-интерфейс)

Откройте новое окно терминала:

```bash
cd /Users/Dom/MyPosSystem/frontend

# Установить зависимости (только первый раз)
npm install

# Запустить dev сервер
npm run dev
```

Frontend будет доступен на: http://localhost:5173

## Интерфейсы системы

Система разделена на **3 независимых интерфейса**:

### 1. Касса (для кассира на планшете)
- **URL:** http://localhost:5173/pos
- Полноэкранный интерфейс
- Выбор товаров, создание заказов
- Оплата наличными/картой
- **БЕЗ** доступа к админке

### 2. Админка (для управляющего с компьютера)
- **URL:** http://localhost:5173/admin
- Управление товарами
- Ингредиенты и техкарты (скоро)
- Настройка цен и категорий
- Профессиональный интерфейс с боковым меню

### 3. Дашборд (для управляющего со смартфона)
- **URL:** http://localhost:5173/dashboard
- Мобильно-оптимизированный интерфейс
- Статистика в реальном времени
- Просмотр заказов
- Автообновление каждые 5 секунд

### Главная страница (только для разработки)
- **URL:** http://localhost:5173/
- Выбор интерфейса
- В продакшене будут отдельные домены

## Первые шаги

### 1. Добавьте товары
1. Откройте http://localhost:5173/admin
2. Нажмите "+ Добавить товар"
3. Создайте несколько товаров с ценами

Примеры:
- Латте - 450₸ (Категория: Напитки)
- Американо - 350₸ (Категория: Напитки)
- Круассан - 300₸ (Категория: Выпечка)
- Чизкейк - 600₸ (Категория: Десерты)

### 2. Протестируйте кассу
1. Откройте http://localhost:5173/pos (Касса)
2. Нажмите на товары для добавления в заказ
3. Выберите способ оплаты (наличные/карта)
4. Создайте заказ

### 3. Посмотрите статистику (с телефона или компьютера)
1. Откройте http://localhost:5173/dashboard
2. Увидите:
   - Количество заказов
   - Общую выручку
   - Разбивку по способам оплаты
   - Топ товаров
   - Список всех заказов

## Основные функции

### Касса (для кассира)
- Просмотр всех товаров
- Фильтр по категориям
- Добавление товаров в заказ
- Изменение количества
- Оплата наличными или картой
- Автоматическое создание чека

### Админка (для управляющего)
- Создание новых товаров
- Редактирование цен
- Управление категориями
- Включение/выключение товаров
- Удаление товаров

### Дашборд (для управляющего)
- Статистика за сегодня
- Автообновление каждые 5 секунд
- Топ продаж
- История заказов
- Разбивка по способам оплаты

## API эндпоинты

Полная документация: http://localhost:8000/docs

Основные:
- `GET /api/products` - список товаров
- `POST /api/products` - создать товар
- `POST /api/orders` - создать заказ
- `GET /api/orders/today` - заказы за сегодня
- `GET /api/orders/stats/today` - статистика за сегодня

---

## База данных

Сейчас используется SQLite (`backend/mypos.db`).

Для перехода на PostgreSQL:
1. Установите PostgreSQL
2. Создайте базу данных
3. Измените `DATABASE_URL` в `backend/.env`

## Проблемы?

### Backend не запускается
```bash
cd backend
source venv/bin/activate
pip install -r requirements.txt
```

### Frontend не запускается
```bash
cd frontend
rm -rf node_modules
npm install
```

### База данных пустая
Backend автоматически создаст таблицы при первом запуске.
Просто добавьте товары через админку.
